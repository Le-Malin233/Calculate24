# Calculate24
A tiny game demo for the classic math game calculate 24. The GUI interface is implemented using FLTK.

There are three modes for the game.

## Calculate Mode
You are required to enter 4 numbers from 1 to 13 (11, 12, 13 can also be replaced by J, Q, K). If there is a solution, the program will randomly generate a calculation formula that meets the conditions. If there is no solution, the program will output "No solution".

## Challenge Mode
There will be 4 numbers on the screen, you are required to enter the correct calculation formula in 1 minutes. You will lose if the answer is wrong or the time is end.

## Infinite Mode
There will be 4 numbers on the screen, you are required to enter the correct calculation formula. You will get 1 point if you are right or the game will end. Best points will be saved as a record.
